# express-bun

Small starter kit for express.js with bun.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

or

```bash
bun run nodemon
```

This project was created using `bun init` in bun v0.2.2. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
